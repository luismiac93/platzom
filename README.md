#Platzom

Platzon es un idioma inventado para el curso de fundamentos de javascript

## Descripción del idioma
-regla uno
-regla 2
## Instalación
```
npm install Platzom
```
## Uso
```
import platzom from 'platzom'

platzom("programar")
```
